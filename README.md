# Telegram Notes Bot

Telegram-бот для создания, просмотра и удаления заметок.  
Проект реализован на Python с использованием Telegram Bot API.

## 🚀 Функциональность

- `/add <текст>` — добавление новой заметки  
- `/list` — просмотр списка всех заметок  
- `/delete <номер>` — удаление заметки по номеру  
- `/help` — список доступных команд  

Бот обрабатывает пользовательский ввод, валидирует данные и корректно работает с хранением информации.

---

## 🛠 Технологии

- Python 3.x
- Telegram Bot API
- Асинхронная обработка команд
- JSON для хранения данных
- Логирование ошибок и действий

---

## 🧠 Архитектура проекта

Проект построен с разделением логики:

- Обработка команд
- Работа с данными
- Взаимодействие с Telegram API
- Обработка ошибок

Структура позволяет легко масштабировать проект и добавлять новый функционал.
